# Prompt Bank

**10 copy-paste prompts for common marketing tasks.** Zero setup required — paste into Copilot Chat, Claude.ai, ChatGPT, or any AI assistant and get results in seconds.

## How to Use

1. Open the prompt file below
2. Copy the text in "The Prompt" section
3. Paste it into your AI assistant
4. Replace the `[brackets]` with your actual content
5. Send

## Tips for All Prompts

- **Be specific with your input.** "Our hackathon had 2,400 registrations, 45% above target" gets better results than "our hackathon went well."
- **Specify the audience.** Adding "this is for my VP" or "this is for the team" dramatically changes the output.
- **Iterate.** If the first output isn't right, say "make it shorter" or "more data-driven" or "less formal."
- **Combine prompts.** Use `data-to-insights` first, then feed the output into `doc-to-slides` for a complete data → presentation pipeline.
