Hey team! I put together a collection of AI prompts and skills that can help with the stuff we do every day — building decks, prepping for meetings, writing emails, analyzing data, and more.

- **Prep for any meeting** with talking points + anticipated Q&A
- **Cut any doc in half** while keeping the important stuff
- **Turn raw data into executive slides** with chart suggestions
- **Pressure-test a plan** before you present it (my personal favorite)

Check it out: https://github.com/akiyaani2/solution-marketing-skills

Start with the `/prompts` folder — everything there is copy-paste ready. If you want to go deeper, there are skills, scheduled automations, and even agent blueprints for Copilot Studio.
